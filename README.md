# mist-fn-python

## prerequisites
1. Python 3.6+ or Python 2.7 and virtualenv

## Installation 
1. git clone this repo
2. `cd mist-fn-python`
3. `virtualenv venv`
4. `source venv/bin/activate`
5. `pip install -r requirements.txt`

## How to run

1. `cd sample_app`
2. Build the artifact by running `python setup.py bdist_egg`
